デジタル出席簿 (Digital Attendance Manager)

ブラウザだけで動作する、シンプルで軽量な月間カレンダー型出席管理ツールです。
学習eポータルやExcelよりも手軽に、直感的に日々の出欠記録を行いたいという教育現場やクラブ活動のニーズに応えるために作成されました。

🌟 主な特徴

インストール不要: HTMLファイルをブラウザで開くだけですぐに使えます。

クリックだけの簡単操作: セルをクリックするごとに「出席(○)」→「遅刻(△)」→「欠席(×)」→「クリア」と切り替わります。

自動保存: データはブラウザ内（LocalStorage）に自動的に保存されます。

リアルタイム集計: 出席・欠席・遅刻の日数が即座に集計されます。

アラート機能: 指定した欠席日数を超えた児童・生徒を赤色でハイライト表示し、指導が必要な対象を可視化します。

CSVエクスポート: 月末の集計用に、Excelでそのまま開ける形式（BOM付きCSV）でデータをダウンロードできます。

レスポンシブ対応: PC、タブレット、スマートフォンなど、デバイスを問わず利用可能です。

🚀 使い方

1. 起動

attendance_manager.html ファイルを、Google Chrome, Edge, SafariなどのWebブラウザにドラッグ＆ドロップして開いてください。

2. 名簿の登録

画面右上の「名簿管理」ボタンをクリックし、児童・生徒の氏名を入力して「追加」してください。

ドラッグ＆ドロップでの並び替えは現在未実装ですが、削除・再登録で整理可能です。

3. 出欠の入力

カレンダーの日付と氏名が交差するセルをクリックして入力します。

1回クリック: <span style="color:green">○ (出席)</span>

2回クリック: <span style="color:orange">△ (遅刻)</span>

3回クリック: <span style="color:red">× (欠席)</span>

4回クリック: 空白 (入力取り消し)

4. 月の切り替え

ヘッダー中央の日付横にある「<」「>」ボタンで、表示する月を切り替えます。過去や未来のデータも入力可能です。

5. アラート設定

画面下部の設定欄で、欠席日数の閾値（デフォルトは3日）を変更できます。この日数以上欠席がある場合、集計欄と氏名が赤く表示されます。

6. データの出力

「CSV出力」ボタンをクリックすると、現在表示されている月のデータをCSVファイルとしてダウンロードできます。

⚠️ 注意事項

データ保存場所: データはブラウザの「ローカルストレージ」に保存されます。

サーバーには送信されません（プライバシー重視）。

ブラウザの履歴やキャッシュを削除すると、データが消える場合がありますのでご注意ください。

別のPCやブラウザで開いた場合、データは共有されません。

💻 開発者向け情報

このアプリは単一のHTMLファイルで構成されています（Single File Application）。
インターネット接続があれば、CDNからライブラリを読み込んで動作します。
オフラインで使用したい場合は、Tailwind CSSとFontAwesomeのライブラリをローカルにダウンロードし、パスを書き換えることで対応可能です。

Created for simple and efficient classroom management.
